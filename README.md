## Getting Started

### Prerequisites

* Windows 10 or 11
* ***For development:***
  * Visual Studio 2022
  * C++20 <sub><sup>&</sup></sub> C17

### Installation

* Clone the repo

## Usage

### To play

1. Open `Release` folder
2. Run `CrossingGame.exe`

### To peruse the code

* Run `CrossingGame.sln`
